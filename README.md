# RBAC Backend with TypeScript

This project is a simple implementation of a Role-Based Access Control (RBAC) system using Node.js and TypeScript. It demonstrates how to manage user roles and permissions in an Express application.

## Features

- Define roles and permissions
- Assign roles to users
- Middleware to protect routes based on permissions

## Prerequisites

- [Node.js](https://nodejs.org/) (v20 or later)
- [npm](https://www.npmjs.com/)

## Setup

1. **Clone the repository:**
